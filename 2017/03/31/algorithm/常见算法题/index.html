有序向量的唯一化（去除重复元素）

低效方法：

逐一比较相邻元素，删除重复中的后者。

对于每一次remove操作都需要对后置位元素进行复制操作，最坏的情况是所有的元素都是相等的，那么时间复杂度将达到(n-2)+(n-3)+···+2+1=o(n^2)

高效方法：

逐一扫描，直至遇到不相同的元素，将其移至紧靠于前者后侧，然后删除多余元素。

时间复杂度为 3*(n-1) = o(n)

由此可见，算法效率大大提升
