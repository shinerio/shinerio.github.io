## 🔧 技术细节

### CSS 变量系统

所有颜色、间距、字号都使用 CSS 变量,易于维护和调整:

```css
:root {
    --color-accent-primary: #FF8C42;
    --font-size-base: 1.125rem;
    --spacing-xl: 2.5rem;
    --radius-lg: 1.25rem;
}
```

### 动画性能优化

- 使用 `transform` 而非 `top/left`
- 使用 `opacity` 过渡
- 硬件加速 (`will-change`)
- 合理的缓动函数

### 可访问性

- 充足的颜色对比度
- 焦点可见状态
- 语义化 HTML
- ARIA 标签
