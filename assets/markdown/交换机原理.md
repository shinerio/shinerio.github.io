# 1. SAT（MAC地址表、FDB、CAM）
交换机的Source Address Table（源地址表），也称为MAC地址表、FDB(forwarding database)或CAM(Content Addressable Memory)，是一种用于存储和管理设备MAC地址和端口之间映射关系的表格。当交换机接收到一个帧（以太网数据包）时，它会查看帧中的源MAC地址，并在源地址表中查找与该MAC地址相关的端口信息。如果表中存在该地址的映射，则交换机将确定目标MAC地址对应端口，并将帧转发到目标端口。如果表中不存在该地址的映射，交换机将广播（flooding）这个帧到所有其他端口，以便学习新的MAC地址与端口的映射关系，并在以后的转发中使用这些信息进行优化。
