## 插件安装

[hexo-blog-encrypt](https://github.com/MikeCoder/hexo-blog-encrypt)

```shell
npm install --save hexo-blog-encrypt
```

## 开启加密插件

```shell
# Security
encrypt: # hexo-blog-encrypt
  enable: true
```

## 给文章加密

```markdown
---
title: Hexo文章加密访问
date: 2019-08-31
password: shinerio   
abstract: hexo文章加密教程
message: 密码：shinerio
---
```

- password: 是该博客加密使用的密码
- abstract: 是该博客的摘要，会显示在博客的列表页
- message: 这个是博客查看时，密码输入框上面的描述性文字
